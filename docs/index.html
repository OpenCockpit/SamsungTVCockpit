SamsungTVCockpit Installation

To install the SamsungTVCockpit plugin execute the following commands in a telnet console on your Enigma2 box:
  • opkg install wget (required the first time only)
  • wget https://OpenCockpit.github.io/SamsungTVCockpit/samsungtvcockpit.sh -O - | /bin/sh
The installation script will also install a feed source that enables a convenient upgrade to the latest version with the following commands or automatically as part of an Enigma2 image upgrade:
  • opkg update
  • opkg upgrade
